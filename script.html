In computing, a script is a relatively short and simple set of instructions that typically automate an otherwise manual process. The act of writing a script is called scripting. Scripting language or script language describes a programming language that it is used for a script.

Scripts are usually interpreted rather than compiled.

Notable environments that are scriptable, can be automated via script, include application software, text editor, web page, operating system, embedded system, and computer game.

Although any language could be used for scripting, in practice languages tend to either be used for scripting or for application development. But, some are used for both.
